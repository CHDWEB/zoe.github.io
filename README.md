# 算术小火箭

这是一个纯静态的 iPad 友好学习小游戏集合，包含数学游戏和英语游戏。

## 运行方式

直接把本目录部署到任意静态站点即可：

- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages

入口文件是 `index.html`，不需要构建步骤，也不需要服务器后端。

## iPad 使用

部署后，在 iPad Safari 打开站点网址。需要像 App 一样使用时，可以点击 Safari 分享按钮，然后选择“添加到主屏幕”。

## 功能

- 首页游戏大厅
- 数学游戏：100 以内两位数不进位、不退位加减法
- 数学游戏：前三关竖式计算，第四关开始横式计算
- 数学游戏：1-6 数独，支持冲突提示和完成检测
- 英语游戏：单词闯关，包含颜色、服饰、文具、月份、运动、球类、季节、天气、动物
- 英语游戏：词汇量超过 8 个的分类，单关内不会重复出同一个英文词
- 英语游戏：句子练习，区分人称、do/don't/does/doesn't 和 like/likes、love/loves、hate/hates
- 英语游戏：支持浏览器英文朗读
- 60 秒挑战
- 大数字键盘，适合触控
- 本地保存当前关卡进度
- PWA 离线缓存
