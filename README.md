# 算术小火箭

这是一个纯静态的 iPad 友好数学小游戏，用于练习 20 以内加减法。

## 运行方式

直接把本目录部署到任意静态站点即可：

- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages

入口文件是 `index.html`，不需要构建步骤，也不需要服务器后端。

## iPad 使用

部署后，在 iPad Safari 打开站点网址。需要像 App 一样使用时，可以点击 Safari 分享按钮，然后选择“添加到主屏幕”。

## 功能

- 10 以内加法
- 10 以内减法
- 20 以内加法
- 20 以内减法
- 20 以内加减混合
- 60 秒挑战
- 大数字键盘，适合触控
- 本地保存当前关卡进度
- PWA 离线缓存
